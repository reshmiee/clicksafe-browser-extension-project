# Landing Page - Marketing Website

## Structure

```
landing-page/
├── index.html             # Main landing page
├── style.css              # Styles
├── script.js              # JavaScript functionality
├── assets/
│   └── images/            # Screenshots, graphics
└── downloads/
    └── clicksafe.zip      # Extension ZIP for download
```

## Setup

1. Build the extension ZIP:
   - Zip the entire `frontend/` folder
   - Place in `downloads/clicksafe.zip`

2. Deploy to Netlify/Vercel:
   - Drag and drop the `landing-page/` folder
   - Site will be live at provided URL
